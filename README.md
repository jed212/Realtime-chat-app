# Realtime-chat-app Project
This project is about a real-time messaging application using Django, Django channels, web-sockets, Django REST framework and React js.

## App Frontend
* React.js: The frontend is built using the React.js library, which is a popular JavaScript library for building user interfaces.
* WebSockets: WebSockets are used for real-time communication on the frontend.
## App Backend
* Django and Django Channels: Django is a Python web framework, and Django Channels is an extension that adds support for real-time functionality using WebSockets.
* Django REST framework: Django REST framework is used for creating RESTful APIs in the backend.
## Database
* PostgreSQL: The project uses the PostgreSQL database for data storage.
